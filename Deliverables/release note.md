# Taxease Release Notes

## Version 1.0.0

### Release Date

June 2026

### Overview

This release introduces the first version of Taxease, a mobile-first tax management platform designed to help Nigerian users calculate and understand their PAYE tax obligations.

### New Features

* Individual PAYE Tax Calculator
* Salary and deduction input forms
* Tax breakdown and net salary estimation
* Batch payroll calculation
* Export to CSV/Excel
* Tax Learning Hub
* Tax Glossary
* Frequently Asked Questions (FAQ)

### User Experience Improvements

* Mobile-first design
* Simplified onboarding process
* Beginner-friendly tax explanations

### Documentation

* README
* User Guide
* FAQ
* Glossary
* Tax Methodology Document

### Known Limitations

* Tax payment integration is not included in this release.
* API integrations are limited to project requirements.
